### Minor Integration Gaps (non-blocking)

1. **ASSET-03: `asset_job_submitted` event never appended**
   - `submitAssetJob` in `packages/engine/src/asset/job-store.ts` inserts the job row but never calls `appendAssetEvent` with type `asset_job_submitted`
   - SSE observers see jobs starting at `active` state, not `submitted`
   - Fix: Add event append after insert

2. **MCP-03: Push notifications not wired**
   - `notifyJobStatusChanged` in `packages/mcp/src/resources/job-status.ts` is exported but never called
   - MCP clients must poll `check-job-status` instead of receiving push updates
   - Functional via polling, but push subscription infrastructure is unused
