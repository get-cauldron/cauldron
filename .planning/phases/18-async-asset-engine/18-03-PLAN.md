The `generateAssetHandler` function (exported for testability):

```typescript
export async function generateAssetHandler({ event, step }: {
  event: { data: { jobId: string; projectId: string } };
  step: { run: <T>(name: string, callback: () => Promise<T>) => Promise<T> };
}): Promise<{ jobId: string; status: string }> {
```

Implementation:
- **Step 1 "submit-to-comfyui"**: Get job from DB. Transition to `'active'`. Call `executor.submitJob({ ...jobParams, jobId })` to get `comfyuiPromptId`. Append `asset_job_active` event. Return `comfyuiPromptId`.
- **Step 2 "poll-completion"**: Poll loop with `const POLL_INTERVAL = 3_000; const TIMEOUT = 300_000;` (per D-17: 5 min default). Use a `while` loop: call `executor.checkStatus(comfyuiPromptId)`. If `done`, return outputs. If elapsed > TIMEOUT, throw `new NonRetriableError('Asset generation timed out after 5 minutes')`. Otherwise `await new Promise(r => setTimeout(r, POLL_INTERVAL))`. This is inside a single `step.run()` — standard `setTimeout` delay, NOT `step.sleep()` (see Research anti-pattern note).
- **Step 3 "collect-artifacts"**: Get the first image from `outputs.images[0]`. Call `executor.getArtifact(outputs, image.filename)` to get `imageBuffer`. Build the `ArtifactSidecar` object with all provenance fields. Call `writeArtifact({ artifactsRoot, jobId, projectId, imageBuffer, imageFilename: image.filename, sidecar })`. Call `completeJob(db, jobId, { artifactPath: dir, outputMetadata })`. Append `asset_job_completed` event.
- **Error handling**: Wrap each step's logic in try/catch. On failure, call `failJob(db, jobId, error.message)`, append `asset_job_failed` event, then rethrow so Inngest handles retry.
