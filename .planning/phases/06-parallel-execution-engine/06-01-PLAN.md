    - Constructor takes `repoPath: string` and optional `binaryPath?: string` (defaults to `process.env['CODEBASE_MEMORY_MCP_BIN'] ?? 'codebase-memory-mcp'`).

    - Private `invoke<T>(tool: string, args: Record<string, unknown>): Promise<T>` method:
      - Builds JSON string via `JSON.stringify({ ...args, project: this.projectName })`
      - Uses `writeFileSync` to write args JSON to a temp file (avoids shell injection per Research open question 1)
      - Calls `execAsync(`${this.binaryPath} cli ${tool} '$(cat ${tmpFile})'`)` OR reads the tmp file approach — safer: write JSON to tmpfile, exec with `cat tmpfile | xargs` approach. Actually simplest safe approach: `execAsync` with the JSON as a properly shell-escaped argument using `JSON.stringify` which produces valid shell-safe strings when single-quote wrapped, BUT handle the edge case of single quotes in the JSON by using `replace(/'/g, "'\\''")`.
      - Parses stdout as MCP envelope: `JSON.parse(stdout)` -> check `isError` -> `JSON.parse(envelope.content[0].text)` -> return as `T`.
    - Public methods (all pass `project` automatically via invoke):

      - `async searchGraph(params: { label?: string; name_pattern?: string; file_pattern?: string }): Promise<GraphSearchResult>` — invoke `'search_graph'` with params
