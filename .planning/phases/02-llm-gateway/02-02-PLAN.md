1. Create `packages/engine/src/gateway/circuit-breaker.ts` per D-17:
   - Define `CircuitState` type: `'CLOSED' | 'OPEN' | 'HALF_OPEN'`
   - Define `ProviderCircuit` interface: `{ state: CircuitState; failureCount: number; lastFailureAt: number; openedAt: number | null }`
   - Constants: `FAILURE_THRESHOLD = 3`, `COOLDOWN_MS = 60_000`, `WINDOW_MS = 120_000`
   - Export `CircuitBreaker` class with:
     - Private `circuits: Map<ProviderFamily, ProviderCircuit>`
     - `isOpen(provider: ProviderFamily): boolean` — returns true if provider is in OPEN state and cooldown has not elapsed. If cooldown elapsed, transitions to HALF_OPEN and returns false. Checks window expiry: if `lastFailureAt` is older than `WINDOW_MS`, resets circuit to CLOSED.
     - `recordFailure(provider: ProviderFamily): void` — increments failureCount. If `failureCount >= FAILURE_THRESHOLD`, sets state to OPEN and records `openedAt = Date.now()`.
     - `recordSuccess(provider: ProviderFamily): void` — resets circuit to CLOSED with failureCount 0.
     - `reset(): void` — clears all circuits (for testing)

   - Export function `filterDiverseModels(modelChain: string[], excludeFamily: ProviderFamily): string[]` — returns only models whose family differs from `excludeFamily`. Used during holdout-stage failover to skip same-family providers.

   - Import types: `PipelineStage, ProviderFamily`
   - Define `classifyError(error: unknown): 'rate_limit' | 'server_error' | 'auth_error' | 'timeout' | 'other'`:
     - If `APICallError.isInstance(error)`: check `error.statusCode` — 429 = rate_limit, 401/403 = auth_error, >=500 = server_error
     - If error has `code === 'ETIMEDOUT'` or `code === 'ECONNABORTED'`: return 'timeout'
     - Otherwise: 'other'
   - Define `shouldRetry(errorKind): boolean` — true for rate_limit, server_error, timeout. false for auth_error and other.
   - Export `executeWithFailover<T>` async function:
     - Parameters: `{ modelChain: string[], stage: PipelineStage, circuitBreaker: CircuitBreaker, implementerFamily?: ProviderFamily, execute: (model: LanguageModel, modelId: string) => Promise<T>, onFailover?: (attempt: FailoverAttempt) => void }`
     - If stage is 'holdout' and `implementerFamily` is provided, filter modelChain through `filterDiverseModels(modelChain, implementerFamily)`. If filtered chain is empty, throw `GatewayExhaustedError` with reason "all models in chain share family with implementer".
     - For each modelId in the (possibly filtered) chain:
       - Check `circuitBreaker.isOpen(getProviderFamily(modelId))` — if open, skip to next model, record a FailoverAttempt with error "circuit breaker open"
       - Try `execute(resolveModel(modelId), modelId)` — on success, call `circuitBreaker.recordSuccess(family)` and return result
       - On error: classify it. If `shouldRetry`: wait `Math.min(1000 * 2^attempt, 8000)` ms (exponential backoff, max 8s) and retry ONCE on the same model (D-14 says one retry). On second failure or non-retryable error, call `circuitBreaker.recordFailure(family)`, record FailoverAttempt, call `onFailover`, continue to next model.
     - If all models exhausted: throw `GatewayExhaustedError(stage, attempts)`
