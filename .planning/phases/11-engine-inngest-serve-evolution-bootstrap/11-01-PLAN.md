Create `packages/api/src/inngest-serve.ts` — a Hono app that serves the `cauldron-engine` Inngest client with all 5 engine functions. Per CLAUDE.md, use Hono (not Next.js Route Handlers) for the backend service agents call into. This eliminates the `ensureInitialized()` complexity from the original plan because `bootstrap.ts` already configures all deps before the server starts.

Implementation for `inngest-serve.ts`:

```typescript
import { serve } from 'inngest/hono';
import { Hono } from 'hono';
import {
  inngest as engineInngest,
  handleBeadDispatchRequested,
  handleBeadCompleted,
  handleMergeRequested,
  handleEvolutionConverged,
  handleEvolutionStarted,
} from '@cauldron/engine';

/**
 * All 5 engine Inngest functions served via the cauldron-engine client.
 * Exported as a constant so the smoke test can verify function count.
 */
export const ENGINE_FUNCTIONS = [
  handleBeadDispatchRequested,
  handleBeadCompleted,
  handleMergeRequested,
  handleEvolutionConverged,
  handleEvolutionStarted,
] as const;

/**
 * Create a Hono app that serves the cauldron-engine Inngest functions.
 * Mount this on the CLI server AFTER bootstrap() has configured all deps.
 * No lazy initialization needed — bootstrap runs first.
 */
export function createInngestApp(): Hono {
  const app = new Hono();

  const handler = serve({
    client: engineInngest,
    functions: [...ENGINE_FUNCTIONS],
  });

  // Mount at /api/inngest so the Inngest dev server discovers engine functions here
  app.on(['GET', 'POST', 'PUT'], '/api/inngest', handler);
  return app;
}
```

Key decisions:
- Use `inngest/hono` adapter (not `inngest/next`) per CLAUDE.md Hono mandate
- Use `inngest as engineInngest` alias to avoid collision with the `inngest` npm package (same pattern as `execution.ts` router)
- Export `ENGINE_FUNCTIONS` array so the smoke test can verify all 5 are registered
- No `ensureInitialized()` needed — bootstrap.ts configures deps before this app is mounted
- Mount at `/api/inngest` path so Inngest dev server can discover the engine app
